# Блогикум

Блогикум — это социальная сеть для публикации личных дневников. Проект предоставляет пользователям возможность создать свою страницу и публиковать на ней посты, делиться мыслями и событиями из своей жизни.

## Описание проекта

Основная цель проекта — дать людям платформу для ведения личных блогов в формате дневника. Каждый пользователь имеет свою отдельную страницу. Проект находится в стадии активной разработки.

## Технологии

*   Язык программирования: Python
*   Веб-фреймворк: Django
*   Для тестирования используется pytest

### Предварительные требования

*   Установленный Python (версия 3.8 или выше рекомендуется)
*   Менеджер пакетов pip

### Инструкция по установке

1.  **Клонируйте репозиторий**
    git clone https://github.com/vi-ktoria/blogicum.git
    cd blogicum
    
3.  **Создайте и активируйте виртуальное окружение**
    # Для Windows
    python -m venv venv
    venv\Scripts\activate
    # Для Linux/macOS
    python3 -m venv venv
    source venv/bin/activate
    
5.  **Установите зависимости**
    pip install -r requirements.txt
    
7.  **Запустите сервер разработки**
    python manage.py runserver
